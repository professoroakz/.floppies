## Deprecation Notices

None currently. All features will be maintained with backwards compatibility.

## Breaking Changes Policy

- Major versions (X.0) may include breaking changes
- Minor versions (1.X) are backwards compatible
- Patch versions (1.1.X) are bug fixes only
- Deprecations announced one version before removal

## Release Schedule

- **Patch releases**: As needed for bugs and security
- **Minor releases**: Quarterly (every 3 months)
- **Major releases**: Yearly or when significant changes warrant
