# Welcome to .floppies

.floppies is your universal development environment setup tool. One command to install everything you need on any platform.

## What is .floppies?

.floppies automates the setup of a complete development environment across macOS, Linux, and Windows. It installs:

- Programming languages (Node.js, Python, Ruby, Go, Rust)
- Editors and IDEs (VSCode, Vim/Neovim)
- Terminal tools (tmux, modern CLI utilities)
- Container tools (Docker, Kubernetes)
- Cloud CLIs (AWS, Azure, GCP)
- Security tools
- And much more...
